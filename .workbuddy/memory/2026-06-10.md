## 数据追溯页空载与噪音波形图

### 完成内容

1. **HistoryViewModel 增加波形图数据生成**
   - 新增 LiveCharts 引用（ISeries, Axis, SkiaSharp 等）
   - 添加 `NoLoadWaveformSeries` / `NoiseWaveformSeries` 等绑定属性
   - `OnSelectedMotorChanged` 回调中根据选中电机数据生成波形
   - 空载电流波形：模拟启动冲击→稳态波动（500采样点，指数衰减+正弦纹波+随机噪声）
   - 噪音波形：正转/反转双通道（低频波动+随机噪声），红色虚线显示阈值上限
   - Y轴自适应范围，X轴统一采样点

2. **HistoryView.xaml 替换占位符**
   - 引入 `lvc` 命名空间（LiveCharts）
   - 两处"空载与噪音波形图加载区..."占位符均替换为实际波形图
   - 上方：空载电流波形（青色线条 + 红色虚线阈值）
   - 下方：噪音波形（橙色正转 + 紫色反转 + 红色虚线阈值）
   - 支持 X 轴缩放（ZoomMode="X"）

### 波形数据算法

- **空载电流**：启动阶段 t<0.05 用指数衰减模拟冲击电流，之后用正弦纹波+高斯噪声模拟稳态
- **噪音**：基于电机正/反转噪音基准值，叠加低频正弦波（模拟机械振动）+ 随机噪声
- 阈值线：空载电流 1.5A，噪音 60dB，红色虚线标注

## 编译错误修复

- **问题**：`SKPathEffect.CreateDash()` 返回 `SkiaSharp.SKPathEffect`，但 `SolidColorPaint.PathEffect` 属性需要 `LiveChartsCore.SkiaSharpView.Painting.Effects.PathEffect` 类型
- **修复**：将 `SKPathEffect.CreateDash(new float[] { 6, 4 }, 0)` 替换为 `new DashEffect(new float[] { 6, 4 }, 0)`
- **新增using**：`LiveChartsCore.SkiaSharpView.Painting.Effects`
- **构建结果**：0 错误，9 警告（均为 NuGet 包兼容性警告，不影响运行）
